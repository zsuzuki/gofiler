# gofiler

Go + [Bubble Tea](https://github.com/charmbracelet/bubbletea) で作ったシンプルなTUIファイラーです。

## Features

- カーソルキーで移動（`↑/↓` で行移動、`←` で親へ、`→` でディレクトリ移動 or ファイル選択トグル）
- `PgUp/PgDn` でページ移動
- 画面上部にカレントディレクトリ表示
- ファイルサイズ・更新日時を表示
- `Space` で選択トグル
- `d` で削除（確認あり）
- `m` でカレントディレクトリへ移動（確認あり、ディレクトリまたぎ対応）
- `c` でカレントディレクトリへコピー（確認あり）
- `r` でリネーム（名前編集後に確認）
- `f` で名前フィルター
- `s` でサイズ以下フィルター、`S` でサイズ以上フィルター
- `x` で各種フィルターを一括解除
- `?` でキーバインド一覧を表示/非表示
- `q` で終了（確認あり）

## Requirements

- Go 1.22+

## Run

```bash
go mod tidy
go run .
```

## Key Bindings

- `↑/↓`: カーソル移動
- `←`: 親ディレクトリへ移動（移動時にフィルタークリア）
- `→`: ディレクトリなら移動 / ファイルなら選択トグル
- `PgUp/PgDn`: ページ移動
- `Space`: 選択トグル
- `d`: 削除
- `m`: 移動
- `c`: コピー
- `r`: リネーム（編集モード）
- `f`: 名前フィルター
- `s`: サイズ `<=` フィルター
- `S`: サイズ `>=` フィルター
- `x`: 各種フィルターを一括解除
- `?`: キーバインド一覧を表示/非表示
- `q`: 終了

## Size Filter Format

サイズ入力は次の形式に対応します。

- 数値のみ（バイト）: `1024`
- 単位付き: `KB`, `MB`, `GB`, `TB`（例: `500KB`, `1.5MB`, `2GB`）
